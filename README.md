# 旅遊套票平台 (純 JavaScript 開發)

- [線上Demo](https://satarkuo.github.io/LetUsTravel/)

- 純 JavaScript 開發，無使用任何框架。
- 本作品採用現成模板實作功能練習，實作以JS邏輯為主，不包含版型切版。

## 功能特色
- 新增旅遊套票功能
- 切換地區查看旅遊套票
- 使用 C3.js 圖表套件渲染套票地區比重，且新增套票後會即時更新圖表。
- axios API資料串接
